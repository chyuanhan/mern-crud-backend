# Order System Backend

A simple place management backend service based on MERN stack.

## Features

- 🔐 User Authentication & Authorization (JWT)
- 📋 Place Management
- 🏷️ User Signup & Login

## Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- RESTful API

## Getting Started

### Prerequisites

- Node.js (version >= 14)
- MongoDB
- npm or yarn

### Installation

1. Clone repository

```bash
git clone https://github.com/chyuanhan/mern-crud-backend.git
cd mern-crud-backend
```

2. Install dependencies

```bash
npm install
```

3. Start the service

```bash
npm start
```
